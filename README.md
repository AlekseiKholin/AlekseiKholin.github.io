# Сайт-визитка Data Analyst

Современный сайт-визитка для дата-аналитика, созданный с использованием HTML, CSS и JavaScript. Оптимизирован для размещения на GitHub Pages.

## 🚀 Функциональность

- **Адаптивный дизайн** - отлично выглядит на всех устройствах
- **Семантическая HTML5 разметка** - лучшая доступность и SEO
- **Современные CSS стили** - градиенты, анимации, параллакс эффекты
- **Интерактивные элементы** - плавная навигация, анимации при прокрутке
- **Форма контактов** - с валидацией и уведомлениями
- **Модальное окно** - для скачивания резюме
- **Портфолио** - с демонстрацией проектов
- **GitHub Pages ready** - простая настройка публикации

## 🚀 Как запустить

### Локальный запуск

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd site
```

2. Откройте файл `index.html` в браузере.

### Размещение на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите файлы проекта:
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```

3. В настройках репозитория включите GitHub Pages:
   - Перейдите в **Settings** → **Pages**
   - Выберите ветку **main**
   - Нажмите **Save**

4. Ваш сайт будет доступен по адресу: `https://<your-username>.github.io/<repo-name>`
