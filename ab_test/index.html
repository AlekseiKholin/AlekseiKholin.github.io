3. Статистический анализ¶

Формулировка гипотез¶

🎯 Нулевая гипотеза (H₀): Конверсия в контрольной и тестовой группах одинакова → Новый дизайн не влияет на конверсию

🎯 Альтернативная гипотеза (H₁): Конверсия в тестовой группе выше → Новый дизайн увеличивает конверсию (односторонний тест)

Расчёт метрик¶

In [96]:
from scipy import stats

# Конверсия по группам
control = df[df['group'] == 'control']['converted']
treatment = df[df['group'] == 'treatment']['converted']

n_control, n_treatment = len(control), len(treatment)
conv_control, conv_treatment = control.mean(), treatment.mean()

# Абсолютный и относительный лифт
abs_lift = conv_treatment - conv_control
rel_lift = (abs_lift / conv_control) * 100

print(f"📊 Конверсия control: {conv_control*100:.2f}%")
print(f"📊 Конверсия treatment: {conv_treatment*100:.2f}%")
print(f"📈 Абсолютный лифт: {abs_lift*100:+.2f} п.п.")
print(f"📈 Относительный лифт: {rel_lift:+.2f}%")

Z-тест для пропорций¶

In [97]:
from statsmodels.stats.proportion import proportions_ztest
# Z-тест (подходит для больших выборок и бинарных исходов)
z_stat, p_value = proportions_ztest(
    count=[treatment.sum(), control.sum()],
    nobs=[len(treatment), len(control)],
    alternative='larger'  # односторонний тест: treatment > control
)

print(f"\n🔬 Z-статистика: {z_stat:.4f}")
print(f"🔬 p-value: {p_value:.4f}")
print(f"✅ Статистическая значимость (α=0.05): {'ДА' if p_value < 0.05 else 'НЕТ'}")

🔬 Z-статистика: 1.6677
🔬 p-value: 0.0477

Доверительный интервал для лифта¶

In [98]:
# Стандартная ошибка разницы пропорций
se_diff = np.sqrt(
    conv_control * (1 - conv_control) / n_control +
    conv_treatment * (1 - conv_treatment) / n_treatment
)

# 95% доверительный интервал
z_critical = 1.96  # для 95% CI
ci_lower = abs_lift - z_critical * se_diff
ci_upper = abs_lift + z_critical * se_diff

print(f"\n🎯 95% CI для абсолютного лифта: [{ci_lower*100:+.2f}%, {ci_upper*100:+.2f}%]")
print(f"🎯 95% CI для относительного лифта: [{(ci_lower/conv_control)*100:+.2f}%, {(ci_upper/conv_control)*100:+.2f}%]")

🎯 95% CI для абсолютного лифта: [-0.38%, +4.78%]
🎯 95% CI для относительного лифта: [-2.68%, +33.38%]

Power analysis (пост-фактум)¶

In [99]:
from statsmodels.stats.power import zt_ind_solve_power
from statsmodels.stats.proportion import proportions_ztest, proportion_effectsize

# Расчёт мощности теста (насколько мы уверены, что не пропустили эффект)
effect_size = proportion_effectsize(conv_treatment, conv_control)
power = zt_ind_solve_power(
    effect_size=effect_size,
    nobs1=n_control,
    alpha=0.05,
    ratio=n_treatment/n_control,
    alternative='larger'
)

print(f"\n⚡ Мощность теста: {power*100:.1f}%")
print(f"💡 Интерпретация: {'✅ Достаточно для обнаружения эффекта' if power >= 0.8 else '⚠️ Риск ложноотрицательного результата'}")

⚡ Мощность теста: 51.0%
💡 Интерпретация: ⚠️ Риск ложноотрицательного результата

# Bootstrap для визуализации неопределённости
def bootstrap_lift(df, n_bootstrap=1000, seed=42):
    np.random.seed(seed)
    lifts = []
    for _ in range(n_bootstrap):
        sample = df.sample(frac=1, replace=True)
        c = sample[sample['group']=='control']['converted'].mean()
        t = sample[sample['group']=='treatment']['converted'].mean()
        lifts.append((t - c) / c * 100)  # relative lift in %
    return np.array(lifts)

bootstrap_lifts = bootstrap_lift(df)

fig = px.histogram(
    x=bootstrap_lifts, nbins=50,
    title='🔍 Bootstrap Distribution of Relative Lift',
    labels={'x': 'Relative Lift (%)', 'y': 'Frequency'},
    marginal='box'
)
fig.add_vline(x=rel_lift, line_dash='dash', line_color='red', annotation_text='Observed')
fig.add_vline(x=0, line_dash='dash', line_color='gray')
fig.update_layout(height=400)
fig.show()

# Процент бутстрап-выборок с лифтом > 0
p_bootstrap = (bootstrap_lifts > 0).mean()
print(f"🎲 Bootstrap p-value: {p_bootstrap:.4f}")

🎲 Bootstrap p-value: 0.9510
